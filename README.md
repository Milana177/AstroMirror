# AstroMirror - Астрологическое зеркало

## Название приложения
**AstroMirror**

## Краткое описание
AstroMirror — это веб-приложение для астрологических практик и самопознания. Пользователи могут получать персонализированные прогнозы (карта дня), строить натальные карты, рассчитывать матрицу судьбы и получать толкования карт Таро. Приложение сохраняет историю запросов и персонализирует контент для авторизованных пользователей.

## Целевая аудитория
- Люди, интересующиеся астрологией, нумерологией и эзотерикой (18–45 лет)
- Пользователи, желающие получать персонализированные астрологические расчеты
- Начинающие астрологи, изучающие натальные карты и матрицу судьбы

## Ключевой функционал
1. Регистрация и авторизация пользователей
2. Карта дня (Таро) — ежедневное предсказание
3. Гадание на трех картах ("Вопрос в колоду")
4. Построение натальной карты
5. Расчет матрицы судьбы (22 аркана)
6. История всех расчетов и гаданий

## Структура приложения (8 разделов)

| № | Раздел | URL | Тип доступа | Описание |
|---|--------|-----|-------------|----------|
| 1 | Главная | `/` | Публичный | Лендинг с описанием сервиса и навигацией |

## Технологический стек

### Frontend
- **Чистый HTML5/CSS3/JavaScript** — приложение не требует сложной реактивности, все расчеты выполняются на клиенте
- **localStorage** — для хранения пользователей, истории и настроек

### Backend
- **Отсутствует (клиент-сайд архитектура)** — все расчеты выполняются на стороне клиента, данные хранятся локально

### База данных
- **localStorage** — используется для хранения всех данных пользователя

### Инфраструктура
- **GitHub Pages / Vercel** — хостинг статического сайта
